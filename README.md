EatAtHome is an Android food ordering application developed in Kotlin.  
The system supports customer food ordering and provides a restaurant admin module for managing menus, orders, and delivery status.

## Features

### Customer Features
- Browse food menus
- Place food orders
- View order status

### Restaurant Admin Features
- Admin dashboard
- Add and manage food menus
- View and update order delivery status
- Restaurant profile management
- Create new admin accounts

## Technologies Used

- Kotlin
- Android Studio
- Firebase

## Notes
